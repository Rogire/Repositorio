Mas oque é programação?

Bom, segundo o Google, programação pode ser definida como um processo de escrita, testes e manutenção de programas de computadores, que por sua vez estão interligados em uma rede maior de instruções de códigos, que são criadas usando Linguagens de programação

Segundo a fonte, linguagens de programação são um método padronizado de comunicar instruções específicas para um computador. Há centenas de variações e todas elas permitem que o programador defina sobre quais dados o computador deverá atuar, como armazenar e transmitir esses dados e quais ações tomar sob variadas circunstâncias.

C

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela Organização Internacional para Padronização (ISO), criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs para desenvolvimento do sistema operacional Unix (originalmente escrito em Assembly).

C++

C++ é uma linguagem de programação compilada multi-paradigma (seu suporte inclui linguagem imperativa, orientada a objetos e genérica) e de uso geral. Desde os anos 1990 é uma das linguagens comerciais mais populares, sendo bastante usada também na academia por seu grande desempenho e base de utilizadores.

Java Script

O JavaScript é uma linguagem de programação nascida em 1995, e que hoje é a linguagem de programação mais popular do mundo. O JavaScript é uma linguagem que é usada tanto no client-side (frontend), tanto no server-side (backend), com o auxílio do Node.js. O JavaScript teve diversas bibliotecas adicionadas a si ao longo dos anos, como o ReactJs e o TypeScript(TS), que auxiliam na programação em JS.
